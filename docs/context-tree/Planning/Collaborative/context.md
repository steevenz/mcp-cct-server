# Topic: Collaborative Planning

## Overview
Collaborative planning defines the interface between the AI's autonomous reasoning and human decision-making. It ensures that the agent remains an assistant, providing transparency and opportunities for intervention during high-stakes planning tasks.

## Key Concepts
- **Human-in-the-Loop (HITL)**: Hard checkpoints where the agent must wait for human clearance.
- **Shared Autonomy**: A model where the human provides high-level goals and the agent proposes detailed sub-goals.
- **Escalation**: Patterns for when the agent should proactively ask for human help.
