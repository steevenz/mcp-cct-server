# Topic: Reasoning Patterns

## Overview
This topic covers the specific reasoning architectures used to coordinate agent behavior. Each pattern offers a different balance between adaptability, transparency, and computational efficiency.

## Key Concepts
- **ReAct (Reason + Act)**: A dynamic loop of thought and action.
- **ReWOO (Reasoning Without Observation)**: An upfront planning approach that minimizes token calls.
- **Tree-of-Thoughts (ToT)**: A branching exploration of reasoning paths.
- **Chain-of-Thought (CoT)**: Linear, step-by-step reasoning transparency.
- **Plan-and-Execute**: Structured decomposition of complex tasks.
