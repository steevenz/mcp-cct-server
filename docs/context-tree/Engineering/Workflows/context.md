# Topic: Engineering Workflows

## Overview
This topic outlines the specialized protocols for executing engineering tasks. It transitions the agent from general reasoning to structured technical implementation.

## Key Concepts
- **Eval-First Execution**: Defining tests and requirements before implementation.
- **15-Minute Unit Decomposition**: Breaking work into agent-sized, independently verifiable units.
- **Complexity Routing**: Matching task difficulty to the appropriate model intelligence tier.
- **Shared Autonomy**: The delegate-review-own coordination model.
