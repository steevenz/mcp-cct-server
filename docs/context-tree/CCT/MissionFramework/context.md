# Topic: Mission Framework

## Overview
The protocol for dispatching and executing focused cognitive missions. It defines the entry points, the operating modes (Human vs Autonomous), and the domain-specific pipelines.

## Key Concepts
- **AGENT_PROMPT (System)**: The global "Constitution" containing identity, arsenal, and core directives.
- **CCT_DISPATCH (Mission)**: Dynamic briefs for specific goals (Coding, Universal) and modes (Human-stop vs Autonomous).
- **The Phase 6.5 Fork**: The critical logic loop defining the level of autonomy for a mission.
