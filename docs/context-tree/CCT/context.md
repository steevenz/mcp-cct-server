# Domain: CCT (Creative Critical Thinking)

## Purpose
The core cognitive framework and reasoning engine of the MCP Server. This domain handles the structured multi-layered thinking pipeline, persisting thoughts into SQLite, and enforcing architectural rigor.

## Scope
Included in this domain:
- Thinking Engines (Primitive, Hybrid, Sequential).
- Cognitive Primitives (22+ modes of thought).
- Persistence logic (SQLite-backed MemoryManager).
- Quality Metrics & Bias Detection.

Excluded from this domain:
- Dashboard UI/UX implementation (handled in Dashboard domain).
- Transport layer specifics (STDIO/SSE).

## Usage
Reference this domain to understand how the AI's "Nervous System" and "Cognitive Exoskeleton" are architected and expanded.
